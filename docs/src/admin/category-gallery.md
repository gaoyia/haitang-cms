# 相册栏目操作说明

相册型栏目复用 **Post + 资源**，无需单独建表。

## 1. 创建相册分类

1. 进入 **内容 → 分类管理**，新建分类。
2. **列表模板**、**详情模板** 均选「相册（图片为主）」。
3. 各语言填写 **SEO 路径** slug（如 `gallery`），公开访问地址为 `/<lang>/categories/gallery`。
4. 保存后可在列表「公开链接」列预览归档页。

## 2. 创建相册文章

1. 新建文章，**分类** 选择上述相册分类。
2. **封面**：选一张图作为相册封面（列表卡片主图、详情页头图）。
3. **附件**：添加多张照片（图片 MIME），详情页以网格展示。
4. **摘要 / 正文**：作为相册说明；gallery 详情页优先展示摘要，正文 Markdown 可选。

## 3. 访客看到的效果

| 页面 | URL | 表现 |
|------|-----|------|
| 相册归档 | `/<lang>/categories/<slug>` | 卡片网格，封面缩略图 + 标题 |
| 相册详情 | `/<lang>/posts/<slug>` | 大图封面 + 附件照片网格 + 说明 |

全站文章列表 `/<lang>/posts` 仍展示所有分类文章，不受分类模板影响。

## 4. 手测清单

- [ ] 分类 slug 中英文均可访问；数字 ID 访问已配置 slug 时 301 到 canonical URL
- [ ] 同一语言 slug 冲突时管理端拒绝、公开页 409
- [ ] gallery 分类下无封面时，列表回退到首张附件图或占位图
- [ ] 草稿 / 定时文章不出现在分类归档 API 与页面中
